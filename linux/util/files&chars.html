  • backup

    $ find . -printf '%m\t%u\t%g\t%p\0' > file.perm

  • restore

    while read -rd $'\0' perms user group file; do
  if [ -e "$file" ]; then
    chown "$user:$group" "$file"
    chmod "$perms" "$file"
  else
    echo "warning: $file not found"
  fi
done < file.perm

Fast copy or moving or someting (Detials -> Brace Expansion)

  • Example 1:
    $ ls | grep foo
$ touch foo{1,2,3}
$ ls | grep foo
foo1
foo2
foo3

  • Example 2

    $ ls | grep foo
$ touch foo-{a..d}
$ ls | grep foo
foo-a
foo-b
foo-c
foo-d

  • Example 3

    $ ls foo-*
foo-a  foo-b  foo-c  foo-d
$ mv foo-{a,}
$ ls foo-*
foo-  foo-b  foo-c  foo-d

  • Example 4

    $ mkdir -p test/{a,b,c,d}
$ tree test/
test/
├── a
├── b
├── c
└── d

4 directories, 0 files

Shell Parameter Expansion

$ x='aabbcc'

$ echo ${x#a*b}
bcc
$ echo ${x#a}
abbcc
$ echo ${x##a}
abbcc

$ echo ${x%b*c}
aab
$ echo ${x%%b*c}
aa

$ echo ${x%c}
aabbc
$ echo ${x%%c}
aabbc
